## About

*Flashcard Touch* features:

- [ ] Manage Cards
- [ ] Manage Decks
- [ ] Manage Folders
- [ ] Learn Cards 
- [ ] Learn Cards like Tinders
- [ ] Sync data for multi devives
- [ ] Import Audio, Image, Search Image
- [ ] Notify user 
- [ ] Track user activity
- [ ] Statistics

## Requirements

* Xcode 8
* iOS 9.0+
